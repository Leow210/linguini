Recommended setup

Picking the Anthropic preset enables native Claude support — your Claude API key works directly (the app translates the OpenAI-format request into Anthropic's schema and back). LM Studio, OpenAI, Google Gemini (via the /v1beta/openai/ compat layer), DeepSeek, Kimi/Moonshot, and OpenRouter all work out of the box.

Common split: a strong API model (Claude / GPT) for lesson generation, a fast local model (LM Studio Gemma) for chat and correction. Plan outline is short, so any model works there.
